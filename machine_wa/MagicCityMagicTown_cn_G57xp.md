# Classic Bank: Magic City
## Physical Setup
-   Level Machine
-   Verify Tilt Sensitivity. If Tilt Ends Game, Note it on Backglass
-   Check coin door killing flippers.
-   Check Flipper Strength. Consider Glo Balls.
## Hardware Setup
-   5-Ball Play.
-   Novelty / No Extra Balls
