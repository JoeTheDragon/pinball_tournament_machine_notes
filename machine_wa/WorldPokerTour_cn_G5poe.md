# World Poker Tour

## Physical Setup

-   Level Machine

-   Verify Tilt Sensitivity

-   Check Flipper Strength

-   Clean Playfield if necessary

-   Clean Glass

-   Verify coin door ball save or no flipper disable when coin door opened.

-   Outlane posts to difficult. Remove rubber on post.
